isHidden: Функция проверяет, скрыт ли элемент. Элемент считается скрытым, если его стиль display равен "none" или если любой из его родительских элементов скрыт.

loadAsyncSrcForTag: Функция предназначена для асинхронной загрузки ресурсов (как изображения или видео) для тегов, указанных в атрибуте anima-src. Это позволяет оптимизировать загрузку страницы, загружая тяжёлые медиа-файлы только тогда, когда они становятся видимыми или необходимыми.

pauseHiddenVideos: Эта функция управляет воспроизведением видео на странице, автоматически приостанавливая видео, которые не видны на экране, и воспроизводя видео, которые должны автоматически играть и становятся видимыми.

loadAsyncSrc: Общая функция для загрузки асинхронных источников для различных тегов, таких как изображения, фреймы и видео.

getTop: Функция для получения вертикальной позиции элемента относительно страницы.

updateOnResize: Функция, вызываемая при изменении размера окна, чтобы повторно загрузить асинхронные ресурсы и обновить элементы в соответствии с новым размером.
